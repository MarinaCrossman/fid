Bring Your Own Laptop Microsoft Word Brochure

Designed a brochure advertising Bring Your Own Laptop's new Microsoft Word Essentials training course. This brochure showcases my knowledge of dividing continuous text into multiple columns, linking images, using watermarks, creating a text wrap, using shapes, pasting an image inside a shape/text, pasting an image within a column of text, creating tables, using master pages, number pages, and use paragraph styles to create a well-balanced design using InDesign. When choosing the colors I used the red, yellow, and blue of the Microsoft logo but instead of using Microsoft's green, I used BYOL's green. I used the Roberto typeface because it is a nice, clean, readable Sans-Serif that comes in a nice variety of weights and thicknesses to use as headings, subs, and body. The audience of the brochure would be BYOL's new and existing customers who are interested in becoming proficient in Microsoft Word.
